Documentação de C++

C++ é uma linguagem de programação de propósito geral, criada por Bjarne Stroustrup em 1983, como uma extensão da linguagem C. Suporta vários paradigmas de programação, incluindo a programação procedural, orientada a objetos e genérica. É amplamente utilizada no desenvolvimento de sistemas, aplicações, jogos, entre outros.

1. Estrutura Básica de um Programa em C++

#include <iostream>  // Biblioteca para entrada e saída de dados

int main() {
    std::cout << "Olá, Mundo!" << std::endl;  // Exibe "Olá, Mundo!" no ecrã
    return 0;  // Indica que o programa terminou corretamente
}

Explicação:

  • #include <iostream>: Inclui a biblioteca de entrada e saída, que permite o uso de std::cout.
  • int main(): Função principal do programa. A execução começa aqui.
  • std::cout: Usado para imprimir mensagens no ecrã.
  • return 0: Indica que o programa terminou com sucesso.

2. Variáveis e Tipos de Dados

Em C++, as variáveis precisam ser declaradas com um tipo de dado. Alguns dos tipos básicos são:

int idade = 25;          // Inteiro
float altura = 1.75f;    // Número decimal (float)
double peso = 70.5;      // Número decimal (double)
char inicial = 'A';      // Caractere
bool isAdulto = true;    // Booleano (verdadeiro ou falso)

3. Estruturas de Controle

3.1 Condicionais

int idade = 20;

if (idade >= 18) {
    std::cout << "É adulto." << std::endl;
} else {
    std::cout << "Não é adulto." << std::endl;
}

3.2 Ciclos

Ciclo for:

for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}

Ciclo while:

int contador = 0;
while (contador < 5) {
    std::cout << contador << std::endl;
    contador++;
}

4. Funções

As funções permitem dividir o código em blocos reutilizáveis.

int soma(int a, int b) {
    return a + b;
}

int main() {
    int resultado = soma(3, 4);
    std::cout << "A soma é: " << resultado << std::endl;
    return 0;
}

5. Programação Orientada a Objetos (POO)

C++ suporta programação orientada a objetos, com conceitos como classes e herança.

5.1 Classes

class Pessoa {
public:
    std::string nome;
    int idade;

    void apresentar() {
        std::cout << "Olá, meu nome é " << nome << " e tenho " << idade << " anos." << std::endl;
    }
};

int main() {
    Pessoa p;
    p.nome = "João";
    p.idade = 30;
    p.apresentar();
    return 0;
}

5.2 Herança

class Animal {
public:
    void som() {
        std::cout << "O animal faz um som." << std::endl;
    }
};

class Cachorro : public Animal {
public:
    void som() {
        std::cout << "O cachorro late." << std::endl;
    }
};

int main() {
    Cachorro c;
    c.som();  // Chama a versão de "som" da classe Cachorro
    return 0;
}

6. Manipulação de Ponteiros

C++ permite a manipulação de ponteiros, que são variáveis que armazenam endereços de memória.

int x = 10;
int* p = &x;  // 'p' armazena o endereço de 'x'

std::cout << "O valor de x: " << *p << std::endl;  // Desreferenciação: acessa o valor apontado por 'p'

7. Bibliotecas Standard (STL)

A Standard Template Library (STL) do C++ inclui estruturas de dados como vector, map, entre outros.

7.1 Vector

#include <vector>
#include <iostream>

int main() {
    std::vector<int> numeros = {1, 2, 3, 4, 5};

    for (int num : numeros) {
        std::cout << num << std::endl;
    }
    return 0;
}

8. Entrada e Saída de Dados

C++ usa cin e cout para entrada e saída de dados.

int idade;
std::cout << "Digite a sua idade: ";
std::cin >> idade;  // Lê a entrada do utilizador
std::cout << "A sua idade é: " << idade << std::endl;

9. Conclusão
