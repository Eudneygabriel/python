COMPUTADOR

Um computador é uma máquina eletrônica que processa dados de forma estruturada, seguindo uma sequência lógica de operações.

O computador é composto por componentes que recebem entradas, processam essas informações e geram saídas úteis; que são:

  • Entrada (Input)
  • Processamento
  • Armazenamento
  • Saída (Output)

Entrada (input) refere-se aos dados ou comandos que um computador recebe de fontes externas para serem processados.

Esses dados podem ser inseridos de várias formas, geralmente por meio de dispositivos de entrada. Exemplos de dispositivos de entrada incluem:

  • Teclado: Para digitação de texto, números e comandos.
  • Mouse: Para selecionar, arrastar e clicar em ícones ou botões na interface gráfica.
  • Microfone: Para capturar som ou comandos de voz.
  • Scanner: Para converter documentos físicos em imagens digitais.
  • Sensores: Usados em dispositivos especializados (como smartphones) para captar informações como temperatura, movimento ou luz.

Processamento refere-se à etapa em que o computador manipula, transforma ou interpreta os dados recebidos na fase de entrada.

Os principais aspectos do processamento incluem:

  • Execução de Instruções
  • Unidade de Controle
  • Unidade Aritmética e Lógica (ALU)
  • Memória de Curto Prazo (RAM)

Armazenamento refere-se à função de um computador de salvar e reter dados, programas e informações para uso futuro, seja temporariamente ou permanentemente.

Existem dois tipos principais de armazenamento:

  • Armazenamento temporário (volátil)
  • Armazenamento permanente (não volátil)

Saída (Output) refere-se aos resultados ou informações processadas que o computador envia para fora, tornando-os acessíveis ao usuário ou a outros dispositivos.

Os dados processados pelo computador podem ser apresentados através dos dispositivos de saída, como:

  • Monitor
  • Impressora
  • Caixas de Som
  • Projetor

Etapas do Processamento de uma Tarefa

1. Entrada (Input) do Usuário:

Ação do Usuário: O usuário clica no ícone do navegador web usando o mouse (dispositivo de entrada).

Sinal para o Sistema Operacional: Esse clique gera um sinal que é enviado ao sistema operacional (SO), indicando que o usuário quer abrir o navegador.

2. Reconhecimento da Ação pelo Sistema Operacional:

Recepção do Comando: O sistema operacional recebe o comando de clique e verifica qual programa está associado a esse ícone, no caso, o navegador web.

Busca pelo Programa no Armazenamento: O SO localiza o arquivo executável do navegador no dispositivo de armazenamento (geralmente no SSD ou HD).

3. Carregamento do Programa para a Memória RAM:

Transferência para a Memória: O sistema operacional carrega o navegador da unidade de armazenamento para a memória RAM (memória de curto prazo). Isso ocorre porque a RAM permite um acesso mais rápido à CPU.

Alocação de Recursos: O SO também aloca os recursos necessários para o navegador, como espaço na memória e permissões de uso da CPU.

4. Execução do Programa pela CPU (Processamento):

Início do Processamento: Com o navegador na RAM, a CPU começa a executar as instruções do programa. A CPU segue instruções como abrir a interface do navegador, carregar componentes como o motor de renderização da página, e preparar o ambiente de navegação.

Interações Constantes: A CPU continua interagindo com a memória RAM, recuperando e processando instruções e dados necessários para que o programa funcione.

5. Renderização da Interface (Saída):

Saída para o Monitor: O navegador carrega e a interface gráfica do programa é exibida no monitor (dispositivo de saída), mostrando a tela inicial ou página em branco do navegador.

Agora, o usuário pode começar a interagir com o navegador, inserindo URLs, clicando em botões, etc., gerando novas entradas.

6. Gerenciamento Contínuo pelo Sistema Operacional:

Gerenciamento de Processos: O SO gerencia o navegador enquanto ele está aberto, alocando recursos da CPU e memória conforme o usuário abre novas abas ou carrega páginas.

Memória Virtual (se necessário): Se a RAM começar a ficar cheia, o sistema pode mover parte dos dados do navegador para o armazenamento secundário (HD ou SSD) usando memória virtual, garantindo que o computador continue a funcionar sem travar.

7. Saída Contínua (Output Dinâmico):

Exibição de Páginas Web: O navegador recebe dados da internet, processa as informações e as exibe na tela como páginas web. Essas páginas, que são renderizadas pelo motor do navegador, são uma forma contínua de saída durante o uso.
