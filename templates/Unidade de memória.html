O que é Memória?

A memória é uma parte fundamental de um computador que armazena dados temporariamente ou permanentemente.

Estrutura da Memória

A memória do computador é organizada em uma estrutura hierárquica que permite o armazenamento e a recuperação de dados de forma eficiente. Essa estrutura inclui diferentes tipos de memória, que são agrupados com base em sua velocidade, capacidade e volatilidade.

  • Memória Primária (RAM e ROM): A memória primária é onde os dados e instruções em uso são armazenados temporariamente. A RAM (Random Access Memory) é volátil e perde os dados quando o computador é desligado, enquanto a ROM (Read-Only Memory) é não volátil e armazena dados permanentemente.
  • Memória Secundária (HDD/SSD): Armazena dados permanentemente e é utilizada para guardar arquivos, programas e o sistema operacional.

Células de Memória

Uma célula de memória é a unidade básica de armazenamento na memória de um computador. Cada célula pode armazenar um bit de informação, que pode ser um 0 ou um 1. As células de memória são organizadas em uma matriz, onde cada célula é endereçada por um número único.

Funcionamento:

  • Leitura: Para acessar os dados em uma célula de memória, o controlador de memória utiliza o endereço da célula para "ler" o valor armazenado.
  • Gravação: Quando novos dados são gravados, o controlador escreve o valor desejado na célula correspondente.

Modos de Endereçamento

Os modos de endereçamento determinam como o endereço da memória é especificado durante a execução de um programa

  • Endereçamento Direto: O endereço da memória é especificado diretamente na instrução.
  • Endereçamento Indireto: O endereço da memória é armazenado em um registrador, e a instrução se refere a esse registrador
  • Endereçamento por Deslocamento: Combina um endereço base com um deslocamento para calcular o endereço final
  • Endereçamento Registrado: Usa um registrador específico como o endereço de memória.

Medindo a Capacidade da Memória

A capacidade da memória é medida em bytes, onde 1 byte é igual a 8 bits. As capacidades podem ser representadas em diferentes unidades:

  • 1 KB = 1.024 bytes
  • 1 MB = 1.024 KB
  • 1 GB = 1.024 MB
  • 1 TB = 1.024 GB

Tipos de Memória

  • Memória RAM (Dinâmica e Estática): Utilizada para armazenar dados temporariamente enquanto o computador está ligado
  • Memória ROM: Armazena firmware e dados permanentes que não mudam.
  • Memória Cache: Uma memória de alta velocidade usada para armazenar dados frequentemente acessados pela CPU.
  • Memória Flash: Usada em dispositivos USB e SSDs, é uma forma de memória não volátil.h
