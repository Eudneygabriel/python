O que é Memória?

Memória em computação é o componente responsável por armazenar dados e instruções que um sistema de processamento (como um computador) precisa para operar. Ela funciona como uma área de armazenamento que pode ser acessada pela CPU para realizar cálculos, executar programas, e gerenciar tarefas.

Estrutura da Memória

A memória do computador é organizada em uma estrutura hierárquica que permite o armazenamento e a recuperação de dados de forma eficiente. Essa estrutura inclui diferentes tipos de memória, que são agrupados com base em sua velocidade, capacidade e volatilidade.

  • A memória primária, ou RAM (Random Access Memory), é essencial para o funcionamento de qualquer computador, pois armazena temporariamente dados e instruções que o processador (CPU) precisa acessar de forma rápida e contínua. Como a RAM é volátil, ou seja, perde todo o seu conteúdo quando o computador é desligado, ela serve apenas para armazenamento temporário. Em outras palavras, os dados são carregados na RAM apenas enquanto o computador está em uso, permitindo a execução eficiente de programas e o acesso rápido a dados.
  • Memória Secundária (HDD/SSD): é usada para armazenamento permanente, conservando os dados mesmo sem energia. Isso inclui dispositivos como discos rígidos (HDDs), unidades de estado sólido (SSDs), pen drives e cartões de memória. Esses dispositivos são fundamentais para manter informações de forma durável, como o sistema operacional, arquivos pessoais, programas instalados e outros dados importantes. A memória secundária, apesar de ser mais lenta que a RAM para acesso direto, é essencial para que os dados e programas sejam mantidos para usos futuros.

Células de Memória

Uma célula de memória é a unidade básica de armazenamento na memória de um computador. Cada célula pode armazenar um bit de informação, que pode ser um 0 ou um 1. As células de memória são organizadas em uma matriz, onde cada célula é endereçada por um número único.

Funcionamento:

  • Leitura: Para acessar os dados em uma célula de memória, o controlador de memória utiliza o endereço da célula para "ler" o valor armazenado.
  • Gravação: Quando novos dados são gravados, o controlador escreve o valor desejado na célula correspondente.

Modos de Endereçamento

Os modos de endereçamento determinam como o endereço da memória é especificado durante a execução de um programa

  • Endereçamento Direto: O endereço da memória é especificado diretamente na instrução.
  • Endereçamento Indireto: O endereço da memória é armazenado em um registrador, e a instrução se refere a esse registrador
  • Endereçamento por Deslocamento: Combina um endereço base com um deslocamento para calcular o endereço final
  • Endereçamento Registrado: Usa um registrador específico como o endereço de memória.

Medindo a Capacidade da Memória

A capacidade da memória é medida em bytes, onde 1 byte é igual a 8 bits. As capacidades podem ser representadas em diferentes unidades:

Byte (B): A unidade básica de armazenamento de dados, capaz de armazenar um único caractere de texto, como uma letra ou número. Equivale a 8 bits.

Kilobyte (KB): Equivalente a 1.024 bytes (embora, em alguns contextos, seja arredondado para 1.000 bytes).

Megabyte (MB): Equivalente a 1.024 kilobytes ou 1.048.576 bytes. Comumente usado para descrever o tamanho de arquivos de mídia e programas.

Gigabyte (GB): Equivalente a 1.024 megabytes ou 1.073.741.824 bytes. Usado para medir a capacidade de memória de computadores e dispositivos.

Terabyte (TB): Equivalente a 1.024 gigabytes ou 1.099.511.627.776 bytes. Utilizado para descrever grandes volumes de armazenamento, como em servidores e data centers.

  • 1 KB = 1.024 bytes
  • 1 MB = 1.024 KB
  • 1 GB = 1.024 MB
  • 1 TB = 1.024 GB

Tipos de Memória

  • Memória RAM (Dinâmica e Estática): Utilizada para armazenar dados temporariamente enquanto o computador está ligado.
  • Memória ROM: Armazena firmware e dados permanentes que não mudam.
  • Memória Cache: Uma memória de alta velocidade usada para armazenar dados frequentemente acessados pela CPU.
  • Memória Flash: Usada em dispositivos USB e SSDs, é uma forma de memória não volátil.
