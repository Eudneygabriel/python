Documentação de Python

Uma linguagem de programação poderosa e de fácil aprendizado

1. Introdução

Python é uma linguagem de programação interpretada de alto nível, conhecida por sua legibilidade e simplicidade. Ela suporta múltiplos paradigmas de programação, incluindo programação orientada a objetos, imperativa e funcional.

2. Instalação

2.1. Requisitos do Sistema

  • Windows, macOS ou Linux
  • Espaço em disco suficiente

2.2. Passos de Instalação

  1. Baixe o instalador do site oficial: python.org
  2. Execute o instalador e siga as instruções na tela.
  3. Verifique a instalação abrindo o terminal e digitando python --version.

3. Sintaxe Básica

Python utiliza indentação para definir blocos de código. Não há necessidade de chaves ou pontos e vírgulas.

def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Mundo")

4. Tipos de Dados

4.1. Números

Python suporta inteiros, flutuantes e números complexos.

a = 10          # Inteiro
b = 3.14         # Flutuante
c = 1 + 2j        # Complexo

4.2. Strings

Strings são sequências de caracteres.

mensagem = "Olá, Python!"
print(mensagem)

4.3. Listas

Listas são coleções ordenadas e mutáveis.

frutas = ["maçã", "banana", "cereja"]
frutas.append("laranja")
print(frutas)

5. Estruturas de Controle

5.1. Condicionais

idade = 18
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

5.2. Laços de Repetição

for i in range(5):
    print(i)

while idade < 21:
    print("Ainda não atingiu 21 anos.")
    idade += 1

6. Funções

Funções são blocos de código reutilizáveis.

def soma(a, b):
    return a + b

resultado = soma(5, 3)
print(resultado)

7. Módulos e Pacotes

Módulos são arquivos Python que contêm definições e implementações. Pacotes são diretórios que contêm múltiplos módulos.

import math

print(math.sqrt(16))

8. Tratamento de Exceções

Exceções são usadas para lidar com erros em tempo de execução.

try:
    resultado = 10 / 0
except ZeroDivisionError:
    print("Erro: Divisão por zero não é permitida.")
finally:
    print("Execução finalizada.")

9. Bibliotecas Padrão

Python vem com uma vasta coleção de bibliotecas padrão para diversas funcionalidades.

  • os: Interação com o sistema operacional
  • sys: Manipulação de variáveis e funções do interpretador
  • datetime: Manipulação de datas e horários
  • json: Trabalhar com dados JSON
