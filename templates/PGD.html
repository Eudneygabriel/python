Documentação Plantaforma de Gerenciamento de Desempregados

Este projeto tem como objetivo desenvolver um plantaforma de controle que permitirá monitorar os desempregados e melhorar . A equipe dividiu as responsabilidades da seguinte forma: um membro ficou responsável pela documentação a ser inserida no HTML, enquanto o outro cuidará do diagrama UML até o dia 10/05/2024.

1. Objetivo do Projeto

Desenvolver um Plantaforma de gerenciamento de desempregados para fornecer dados estatísticos e facilitar o controle das informações relacionadas ao desemprego.

2. Tecnologia Utilizada

Utilizaremos Python como linguagem de programação principal devido à sua versatilidade e ampla gama de bibliotecas. Implementaremos operações CRUD (Create, Read, Update, Delete) para gerenciar os dados dos desempregados de forma eficiente.

3. Modelagem de Dados

Cada desempregado terá os seguintes atributos:

  • Nome
  • Idade
  • Cidade
  • Área de Especialização
  • Experiência
  • Habilidades
  • Email
  • Nº Telemovel

4. Persistência de Dados

Os dados dos desempregados serão armazenados em um banco de dados integrado ao Python.

5. Testes

Serão desenvolvidos testes para garantir que todas as operações CRUD funcionem conforme o esperado, assegurando que os dados sejam armazenados e recuperados corretamente.

6. Responsáveis

  • Membro 1: Responsável pela documentação e inserção no HTML.
  • Membro 2: Responsável pelo diagrama UML.

7. Cronograma

O diagrama UML deve ser finalizado até o dia 10/05/2024, enquanto a documentação será atualizada conforme o andamento do projeto.
