Documentação Plataforma de Gerenciamento de Desempregados

Este projeto tem como objetivo desenvolver uma plataforma de controle que permitirá a monitoração dos desempregados e empresas. O trabalho não foi dividido especificamente entre os membros; foi feito de forma simultânea, com cada um contribuindo na parte onde se sentia mais confortável.

1. Objetivo do Projeto

Desenvolver uma plataforma de gerenciamento de desempregados para fornecer dados estatísticos dos desempregados cadastrados e das empresas, além de facilitar o processo na busca de empregos.

2. Tecnologia Utilizada

Para o projeto, usaremos as seguintes linguagens de programação (Python, Java e C++) e como linguagem de marcação (HTML).

3. Modelagem de Dados

Cada desempregado terá os seguintes atributos:

  • Nome
  • Cidade
  • Área de formação
  • Tempo de experiência
  • Email
  • Nº Telemóvel

4. Persistência de Dados

Os dados dos desempregados serão armazenados em um banco de dados integrado ao Python.

5. Testes

Serão desenvolvidos testes para garantir que todas as operações CRUD funcionem conforme o esperado, assegurando que os dados sejam armazenados e recuperados corretamente.

6. Responsáveis

  • Membro 1: Responsável pela documentação e pelo HTML.
  • Membro 2: Responsável pelo diagrama App.py Python e alguns HTML.
